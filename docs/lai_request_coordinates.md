# Pedido LAI – Coordenadas Geográficas das ERBs (ANATEL/STEL)

## Como submeter / How to submit

**Portal:** [Fala.BR – Plataforma Integrada de Ouvidoria e Acesso à Informação](https://falabr.cgu.gov.br/)

1. Acessar → "Acesso à Informação" → "Registrar Pedido"
2. Órgão destinatário: **ANATEL – Agência Nacional de Telecomunicações**
3. Copiar o texto da seção "Pedido" abaixo
4. Categoria: Telecomunicações → Serviços de Telecomunicações
5. Enviar

## Pedido (Texto oficial em Português)

**Assunto:** Solicitação de coordenadas geográficas (latitude/longitude) das Estações Rádio Base (ERBs) licenciadas no Brasil

**Texto:**

Prezada Agência Nacional de Telecomunicações – ANATEL,

Com fundamento na Lei nº 12.527/2011 (Lei de Acesso à Informação – LAI) e no Decreto nº 7.724/2012, venho, respeitosamente, solicitar o fornecimento das **coordenadas geográficas (latitude e longitude)** de todas as Estações Rádio Base (ERBs) licenciadas e registradas no Sistema de Gestão de Licenciamento de Estações de Telecomunicações (STEL).

### Dados solicitados

Para cada estação licenciada, solicito os seguintes campos:

| Campo | Descrição |
|-------|-----------|
| **NumEstacao** | Número identificador da estação |
| **Latitude** | Coordenada geográfica (graus decimais ou GMS) |
| **Longitude** | Coordenada geográfica (graus decimais ou GMS) |
| **Município** | Município de instalação |
| **UF** | Unidade Federativa |
| **Prestadora** | Operadora responsável |
| **Tecnologia/Tipo** | Tecnologia de acesso (2G/3G/4G/5G/NB-IoT) |
| **Faixa de Frequência** | Frequência de operação (MHz) |
| **Altura da Antena** | Altura da estrutura/antena (metros) |
| **Potência de Transmissão** | EIRP ou potência do transmissor |
| **Data de Licenciamento** | Data de emissão da licença |
| **Situação** | Status da licença (ativa, desativada, etc.) |

### Justificativa

1. **As coordenadas geográficas constam no cadastro STEL.** O sistema STEL exige, no processo de licenciamento, o preenchimento obrigatório das coordenadas de instalação de cada ERB (campos Latitude e Longitude). Essas informações existem na base de dados da ANATEL, porém **não foram incluídas** no conjunto de dados disponibilizado via LAI anteriormente (protocolo de pedido anterior que forneceu apenas: NumCnpjCpf, Prestadora, NumEstacao, CodEquipamentoTransmissor, Fabricante, CN, Município, UF).

2. **Interesse público e transparência.** A localização geográfica das ERBs é informação de natureza pública, essencial para:
   - Planejamento de cobertura de telecomunicações em áreas rurais e remotas
   - Estudos acadêmicos sobre infraestrutura de telecomunicações
   - Análise de cobertura de sinal e identificação de lacunas (gaps) de conectividade
   - Políticas públicas de universalização de acesso à internet
   - Verificação de cumprimento de obrigações de cobertura pelas prestadoras

3. **Não há sigilo aplicável.** As coordenadas de instalação de ERBs não configuram informação sigilosa, nos termos dos arts. 23 e 24 da LAI, nem informação pessoal (art. 31). A localização de infraestrutura de telecomunicações já é parcialmente divulgada em bases internacionais (OpenCelliD, Mozilla Location Service) com contribuição colaborativa. A divulgação oficial pela ANATEL asseguraria maior precisão e completude.

4. **Precedentes internacionais.** Agências reguladoras de diversos países disponibilizam publicamente as coordenadas de suas estações licenciadas:
   - **FCC (EUA):** Antenna Structure Registration (ASR) com coordenadas completas
   - **Ofcom (Reino Unido):** Sitefinder database com localizações de ERBs
   - **ACMA (Austrália):** Register of Radiocommunications Licences com coordenadas
   - **CRTC/ISED (Canadá):** Spectrum Management System com coordenadas públicas

### Formato preferencial

Solicito o fornecimento dos dados preferencialmente em formato **CSV (valores separados por vírgula)** ou **XLSX (planilha Excel)**, contendo todos os registros ativos no STEL, com as coordenadas em **graus decimais** (ex.: -23.550520, -46.633309).

Caso o volume de dados exceda os limites de envio do sistema Fala.BR, aceito o fornecimento via link para download (ex.: Google Drive, OneDrive, ou servidor FTP da ANATEL).

### Prazo

Nos termos do art. 11, §1º, da Lei nº 12.527/2011, o prazo para resposta é de **20 (vinte) dias corridos**, prorrogáveis por mais 10 dias mediante justificativa expressa.

Agradeço antecipadamente pela atenção e fico à disposição para eventuais esclarecimentos.

Atenciosamente,
Daniel

## English Translation (for reference only)

**Subject:** Request for geographic coordinates (latitude/longitude) of licensed Base Transceiver Stations (ERBs) in Brazil

Dear National Telecommunications Agency – ANATEL,

Pursuant to Law No. 12,527/2011 (Freedom of Information Act – LAI) and Decree No. 7,724/2012, I respectfully request the provision of **geographic coordinates (latitude and longitude)** for all Base Transceiver Stations (ERBs) licensed and registered in the Telecommunications Station Licensing Management System (STEL).

### Requested data

For each licensed station, I request the following fields:

| Field | Description |
|-------|-------------|
| **NumEstacao** | Station identifier number |
| **Latitude** | Geographic coordinate (decimal degrees or DMS) |
| **Longitude** | Geographic coordinate (decimal degrees or DMS) |
| **Município** | Installation municipality |
| **UF** | State |
| **Prestadora** | Responsible operator |
| **Tecnologia/Tipo** | Access technology (2G/3G/4G/5G/NB-IoT) |
| **Faixa de Frequência** | Operating frequency (MHz) |
| **Altura da Antena** | Antenna/structure height (meters) |
| **Potência de Transmissão** | EIRP or transmitter power |
| **Data de Licenciamento** | License issuance date |
| **Situação** | License status (active, deactivated, etc.) |

### Justification

1. **Geographic coordinates exist in STEL.** The STEL system requires mandatory completion of installation coordinates (Latitude and Longitude fields) during the licensing process. This information exists in ANATEL's database but **was not included** in the dataset previously provided via LAI (which contained only: NumCnpjCpf, Prestadora, NumEstacao, CodEquipamentoTransmissor, Fabricante, CN, Município, UF).

2. **Public interest and transparency.** The geographic location of ERBs is public information, essential for:
   - Telecommunications coverage planning in rural and remote areas
   - Academic studies on telecommunications infrastructure
   - Signal coverage analysis and connectivity gap identification
   - Public policies for universal internet access
   - Verification of operator coverage obligations

3. **No applicable confidentiality.** ERB installation coordinates do not constitute classified information under Arts. 23 and 24 of the LAI, nor personal information (Art. 31). Telecommunications infrastructure locations are already partially disclosed in international databases (OpenCelliD, Mozilla Location Service) via crowdsourcing. Official disclosure by ANATEL would ensure greater accuracy and completeness.

4. **International precedents.** Regulatory agencies in several countries publicly provide licensed station coordinates:
   - **FCC (USA):** Antenna Structure Registration (ASR) with full coordinates
   - **Ofcom (UK):** Sitefinder database with ERB locations
   - **ACMA (Australia):** Register of Radiocommunications Licences with coordinates
   - **CRTC/ISED (Canada):** Spectrum Management System with public coordinates

### Preferred format

I request the data preferably in **CSV** or **XLSX** format, containing all active STEL records, with coordinates in **decimal degrees** (e.g., -23.550520, -46.633309).

If data volume exceeds Fala.BR upload limits, I accept delivery via download link (e.g., Google Drive, OneDrive, or ANATEL FTP server).

### Response deadline

Per Art. 11, §1 of Law 12,527/2011, the response deadline is **20 business days**, extendable by 10 additional days with express justification.

Thank you in advance for your attention. I remain available for any clarifications.

Best regards,
Daniel
